# APK Download Directory

Place your Android APK files here for download.

## Instructions:

1. Build your Android app using Android Studio
2. Copy the generated APK file (usually `app-debug.apk` or `app-release.apk`) to this directory
3. Rename it to `app-debug.apk` for the download link to work

## Current APK Files:
- `app-debug.apk` - The debug APK that will be served for download

## Download URL:
- Main app: `http://127.0.0.1:8000/download/apk/`
- Dashboard app: `http://127.0.0.1:8006/download/apk/` (if configured)

The download link is available in the navigation menu under "System" > "Download APK".
